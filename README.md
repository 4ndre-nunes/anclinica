# anclinica
AN Clinica é um Sistema de Gestão e Marcação de Consultas esenvolvida para otimizar o fluxo de trabalho em clínicas e consultórios. O sistema centraliza o cadastro de pacientes, médicos, especialidades e o histórico de consultas, reduzindo falhas de agendamento e melhorando a experiência do paciente.

# 🚀 Funcionalidades

* Área do Paciente

Cadastro e Login: Crie sua conta para acessar o sistema.
Agendamento Online: Visualize horários disponíveis e marque sua consulta.
Histórico de Consultas: Acompanhe consultas passadas e futuras.
Cancelamento/Reagendamento: Gerencie suas marcações de forma flexível.

* Área da Clínica/Médico

Gestão de Agenda: Visualize a agenda diária, semanal ou mensal.
Gestão de Pacientes: Cadastro detalhado de pacientes.
Confirmação de Consulta: Valide presença e evite faltas.
Dashboard: Visão geral dos agendamentos e status (Confirmado, Aguardando, Cancelado).
